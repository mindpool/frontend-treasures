# 总览

前端工程化是本书的重中之重，也是前端开发中非常重要的一部分，将会用到很多工具和技术，并长时间更新。

## 效能工具 Efficiency Tools

效能工具是指能够帮助提高工作效率、简化开发流程的工具。

在前端开发领域，这些工具包括但不限于：

### 构建工具

- Webpack
- Rollup
- Vite

### 包管理工具

- npm
- yarn
- pnpm

### 代码质量工具

- ESLint
- Prettier
- TypeScript
- Biome

### 性能分析工具

- Chrome DevTools


## 什么是前端工程化 Frontend Engineering

前端工程化是通过工程化的手段和方法，来提高前端开发的效率和质量的一种系统化解决方案。

1. 模块化
2. 组件化
3. 自动化
4. 规范化（代码规范、提交规范、分支规范等）
5. 性能优化
6. 开发环境与部署

主要的工程化目标：

- 提高开发效率
- 降低开发成本
- 保证代码质量
- 提升用户体验
